## 📞 IF YOU GET STUCK

**Can't solve a problem?**

1. Read the approach explained in the guide
2. Trace through the example step-by-step
3. Try solving a simpler version first
4. Come back later with fresh mind

**Understand concept but can't code?**

1. Write pseudocode first
2. Code one line at a time
3. Test after each line
4. Compare with guide solution

**Running out of time?**

1. Focus on high-frequency topics
2. Skip LOW-frequency ones (Geometry, Pipes)
3. Do mini-mock anyway (partial is fine)
4. Speed comes with practice
