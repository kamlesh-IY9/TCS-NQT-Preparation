## PART A: WHEN YOU GET STUCK ON APTITUDE QUESTION

### Situation 1: Mind Goes BLANK (Panic Mode)

**IMMEDIATE ACTION (5 seconds):**

```
STOP solving → Take 3 deep breaths → Clear your mind
Say to yourself: "I've seen similar questions before"
Read the question again slowly (don't panic read)
```

**IF YOU STILL BLANK (20 seconds):**

**DO NOT** waste more time here. Move on.

→ **MARK FOR LATER** (if exam allows)
→ **GUESS intelligently** (see below)
→ **CONTINUE to next question**

---

### Situation 2: You're Calculating But Getting Wrong Answers

**Signs:**
- You've calculated 3 times, getting different answers each time
- You don't trust your calculation
- Time is running out

**WHAT TO DO:**

**Step 1: Stop calculating (20 seconds max spent)**
```
Trying 4th time won't help. You're in a mental loop.
This is a sign: MOVE ON
```

**Step 2: Make an educated guess**
```
Look at answer choices:
- Eliminate obviously wrong ones (too large/too small)
- Pick the middle value if unsure
- Pick most common answer choice
```

**Step 3: Mark & move (Save 5+ minutes)**
```
You save 5 minutes = 4 more questions attempted
Even 1 extra question correct = +5%
```

---

### Situation 3: You Don't Know the Approach

**Signs:**
- You read the question but don't know where to start
- No formula comes to mind
- You're staring at the question blankly

**WHAT TO DO:**

**Step 1: Try elimination (30 seconds)**
```
Numerical questions:
- If answer is percentage, eliminate if >100%
- If answer is cost, eliminate if negative
- Eliminate obviously absurd options

Reasoning questions:
- If asking "who is X", eliminate clearly wrong relations
- If asking pattern, eliminate if doesn't follow trend

Verbal questions:
- Read each option and eliminate most wrong
- Usually 2 are obviously wrong, leaving 2 choices
```

**Step 2: Educated guess from remaining (10 seconds)**
```
If 2 options left: 50% chance either way
If 3 options left: pick middle or most likely
Pattern: Usually answer is not option A or D
Go with gut feeling
```

**Step 3: Move on (DON'T spend >1 min total)**
```
You've tried your best. Next question!
```

---

### Situation 4: You Understand But Time is Running Out

**Signs:**
- Question is partially solved
- You're 30 seconds away from answer
- But timer shows you have 20 questions left with 5 minutes

**WHAT TO DO:**

**Option A: Quick educated guess**
```
If you're close to answer:
- Estimate the final calculation
- Round to nearest option
- Mark and move
```

**Option B: Skip entirely**
```
If you've spent >1.5 min on this question:
- Leave it
- Don't guess
- Get points where you're confident
```

**Time rule:** 1.25 min per question
- If you've spent 1.5 min → you're already over
- Move to easier questions of same section

## PART A: SECTION-SPECIFIC SURVIVAL TRICKS

### NUMERICAL ABILITY TRICKS (When stuck)

**If stuck on Percentage:**
```
Just assume base = 100
Then percentage becomes the number itself
Example: "25% of X is 50"
→ Assume X = 100, then 25 is the answer option
Check if 25 appears in options → likely correct!
```

**If stuck on Profit & Loss:**
```
Assume CP = 100
Then formula becomes simple arithmetic
P&L% = profit number directly
Example: If P&L% = 20, profit = 20 (since CP=100)
```

**If stuck on Time-Speed-Distance:**
```
Pick a simple number as distance
Example: Distance = 100 km
Then calculate time for both speeds
Compare ratios
This often makes calculation easier
```

**If stuck on Data Interpretation:**
```
Look for patterns in given data:
- Largest number usually = answer for "maximum" questions
- Smallest number = answer for "minimum"
- Middle value often = answer for "average"
Percentage questions: focus on proportion, not exact numbers
```

**Critical Hack: Calculator Approach**
```
If your calculation differs from options:
Recalculate ONE time only
If still wrong, pick option closest to your answer
Often "closest to your calculation" = correct
```

---

### REASONING ABILITY TRICKS (When stuck)

**If stuck on Seating Arrangement:**
```
Draw ALL possibilities quickly in 10 seconds
You don't need perfect diagram
Just enough to see the pattern
If still confused, mark as skip
This section has many questions
Skip 1-2, get others = better strategy
```

**If stuck on Blood Relations:**
```
Quick trick: Count generations
If question asks great-grandfather → 3+ generations
Usually answer requires opposite gender
Example: "Brother's father's sister" = Aunt (nearly always)
Pattern: The more steps, the simpler the relation usually is
```

**If stuck on Number Series:**
```
Last resort pattern check (10 seconds):
- Add 1, 2, 3, 4... (Fibonacci-like)?
- Multiply by 2, 3, 4... ?
- Perfect squares: 1, 4, 9, 16...?
- Primes: 2, 3, 5, 7, 11...?
If one of these matches → likely correct!
```

**If stuck on Syllogisms:**
```
Use the contrapositive rule:
"All A are B" → opposite is "No B are NOT A"
This often makes the logic clear
If still unsure: Statement & Conclusion types
These are most predictable. Use logic elimination.
```

**If stuck on Coding-Decoding:**
```
Check if it's:
1. Letter shift (A→B, B→C): How much? Try +1, +2, +3
2. Reverse (LOGIC → CIGOL): Try reversing word
3. First+Last swap: Try swapping first/last letters
One of these 3 usually works!
Try each in 10 seconds, pick most logical
```

### VERBAL ABILITY TRICKS (When stuck)

**If stuck on Reading Comprehension:**
```
You have 2 passages × 4-5 questions each = 8-10 questions
You CANNOT skip all RC. But you can skip 1 passage.

Strategy:
- Read first passage completely
- If second passage is complex/long, skip its 4-5 questions
- Attempt at least 4-5 RC questions from first passage
- This ensures some RC points

Within passage:
- Main idea = usually first/last sentence
- Tone = is author positive/negative/neutral?
- Inference = use only passage info, not your knowledge
```

**If stuck on Error Identification:**
```
Last resort: Look for obvious common errors
(you studied 7 common errors)

Quickly scan:
1. Is subject & verb matching? (Most common error)
2. Are tenses consistent?
3. Any double negatives ("don't have nothing")?
4. Wrong preposition ("different than" instead of "from")?

If you spot ONE of these → it's likely the error!
If you don't spot anything → option D (no error)
```

**If stuck on Sentence Completion:**
```
Clue is usually in the blank:
- "Although" → expect opposite (if positive word before, negative after)
- "Similarly" → expect same (if positive word before, positive after)
- "However" → expect contrast

Use logical connectors to guess!
Even if you don't know the vocabulary,
the logic can guide you
```

**If stuck on Synonyms/Antonyms:**
```
Memory hack: Think of ROOT word
Example: "Benevolent" = "bene" (good) + "volent" (will)
= Good will toward others = Kind, Generous

Even if you don't know exact word:
- Think of root meaning
- Pick most similar option
- You'll be right 60% of time even if unsure!
```

### Situation 3: You Get Wrong Answer on Test Cases

**Signs:**
- Your code runs
- But output doesn't match expected
- You tried fixing once, still wrong
- Time is limited

**WHAT TO DO:**

**Time spent on debugging:** >2 min?
```
STOP. Submit partial solution.
Don't waste more time here.
Better to submit something + get other questions
than submit nothing because you wasted 10 min fixing.
```

**Quick fix checklist (1 min):**
```
1. Off-by-one error? (array index starting from 0)
   int example: for i in range(len(arr)-1) not range(len(arr))

2. Edge case? (empty array, single element, duplicates)
   Add: if len(arr) <= 1: return arr

3. Variable type? (int vs string, list vs array)
   Quick: convert explicitly str(x), int(x)

4. Logic reversal? (did you check < instead of >?)
   Quick: flip one comparison operator

Found fix? Apply and retest (30 sec)
Still wrong? SUBMIT CURRENT → MOVE ON
```

**Important:** In coding section, you get partial credit for:
- ✓ Correct logic (even if code has minor bugs)
- ✓ Handling most test cases (even if 1-2 fails)
- ✓ Code clarity (well-commented code = partial points)

**So submit SOMETHING partial over submitting nothing!**

## EMERGENCY MENTAL BLOCK RECOVERY

### If EVERYTHING Goes Blank (Worst Case)

**The "Reset Button" Technique (Works 80% of time)**

**Step 1: Physical reset (30 seconds)**
```
Stop typing/writing immediately
Close your eyes
Take 5 deep breaths: In (4 count) → Hold (4) → Out (4)
Stretch arms up (2 seconds)
Shake hands (2 seconds)
Open eyes
Read the question again from START
```

**Step 2: Mental reset (20 seconds)**
```
Tell yourself: "I've studied this topic. I know it."
Recall ONE similar problem you solved
Remember: You scored 70%+ on practice, remember?
This is just another problem, not different
Attack it with fresh mind
```

**Step 3: Simplify (30 seconds)**
```
If it's math problem: Assume base = 100 or simple numbers
If it's reasoning: Draw simple diagram or write relationships
If it's coding: Write skeleton code structure first
Simplification often unlocks the block!
```

**Step 4: If still blocked (10 seconds decision)**
```
You've tried 90 seconds
If still nothing comes → MOVE ON
Guaranteed: moving on and coming back later = better result
Block dissolves when you're relaxed, not forced
```
