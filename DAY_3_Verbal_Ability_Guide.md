## SECTION 2: ERROR IDENTIFICATION (45 MINUTES)

**Frequency in TCS NQT:** 4-5 questions per paper

### COMMON ERRORS TO LOOK FOR

**1. Subject-Verb Disagreement**
- "The team are playing" → should be "is playing" (team is singular)
- "Each of the students have completed" → should be "has completed"

**2. Tense Inconsistency**
- "He went to the store and buys milk" → should be "bought"
- "She has been working there for five years and will quit next month" → OK (different timelines)

**3. Pronoun Errors**
- "Everyone has submitted their reports" → should be "his or her reports" or rephrase
- "Neither John nor Mary have completed it" → should be "has completed"

**4. Article Errors**
- "He is teacher" → should be "a teacher"
- "The sun is bright" → OK (specific)

**5. Preposition Errors**
- "Depend on" (not "of")
- "Divided among" (many things), "divided between" (two things)
- "Different from" (not "different than")

**6. Comma Splice / Run-on**
- "He likes reading, he doesn't like writing" → should be "and he doesn't" or semicolon

**7. Double Negative**
- "I don't have nothing" → should be "I don't have anything"

---

### EXAMPLE 1 (Subject-Verb)

**Sentence:** "The committee have decided to postpone the meeting."

**Error?** YES - "committee" is singular, so should be "has decided" not "have decided"

**Corrected:** "The committee has decided to postpone the meeting."

---

### EXAMPLE 2 (Tense)

**Sentence:** "When he arrived at the office, he finds his desk messy."

**Error?** YES - Tense shift: "arrived" (past) but "finds" (present)

**Corrected:** "When he arrived at the office, he found his desk messy."

---

### EXAMPLE 3 (Preposition)

**Sentence:** "Different than his brother, Ajay is very punctual."

**Error?** YES - Should be "different from" not "different than"

**Corrected:** "Different from his brother, Ajay is very punctual."

---

### EXAMPLE 4 (TCS-Level)

**Sentence:** "The manager as well as the employees has decided to work on Saturday."

**Error?** Actually NO error in many guidelines, though some prefer "have."
- With "as well as," the verb agrees with the first noun
- "Manager" is singular, so "has" is technically correct

But this is tricky - the meaning suggests plural, so safer to rephrase as:
"The manager and the employees have decided..."

### EXAMPLE 1 (Basic)

**Sentence:** "Although he studied hard, he _________ the exam."

A) passed
B) failed
C) excelled
D) dominated

**Answer: B) failed**

Clue: "Although" shows contrast. If he studied hard, we'd expect "passed," so the contrast means he failed.

---

### EXAMPLE 2 (TCS-Level)

**Sentence:** "The scientist's _________ research finally _________ the long-standing theory."

A) meticulous, contradicted
B) careless, supported
C) preliminary, confirmed
D) extensive, disproved

**Answer: A) meticulous, contradicted**

Logic: "finally disproved" or "contradicted" suggests overturning a theory. For this to be significant, the research should be "meticulous" (careful, thorough).

---

### EXAMPLE 3 (TCS-Level - Double Blank)

**Sentence:** "Her _________ attitude masked her _________."

A) cheerful, sadness
B) cheerful, happiness
C) serious, confidence
D) calm, anxiety

**Answer: A) cheerful, sadness**

Clue: "masked" suggests hiding something opposite. A cheerful attitude masking sadness makes sense.

### SIMPLE CODING PROBLEM 1: SUM OF ARRAY

**Problem:** Given an array, find the sum of all elements.

**Input:** arr = [1, 2, 3, 4, 5]
**Output:** 15

**Solution:**
```python
def sum_array(arr):
    total = 0
    for num in arr:
        total += num
    return total

# Or simpler
def sum_array(arr):
    return sum(arr)
```

**Time Complexity:** O(n)
**Space Complexity:** O(1)

---

### SIMPLE CODING PROBLEM 2: FIND MAXIMUM

**Problem:** Find the maximum element in an array.

**Input:** arr = [3, 7, 2, 9, 5]
**Output:** 9

**Solution:**
```python
def find_max(arr):
    if len(arr) == 0:
        return None

    max_val = arr[0]
    for num in arr:
        if num > max_val:
            max_val = num
    return max_val

# Or simpler
def find_max(arr):
    return max(arr)
```

**Time Complexity:** O(n)
**Space Complexity:** O(1)

---

### SIMPLE CODING PROBLEM 3: REVERSE ARRAY

**Problem:** Reverse an array.

**Input:** arr = [1, 2, 3, 4, 5]
**Output:** [5, 4, 3, 2, 1]

**Solution:**
```python
def reverse_array(arr):
    return arr[::-1]  # Python slicing

# Or manually
def reverse_array(arr):
    left, right = 0, len(arr) - 1
    while left < right:
        arr[left], arr[right] = arr[right], arr[left]
        left += 1
        right -= 1
    return arr
```

**Time Complexity:** O(n)
**Space Complexity:** O(1) for manual; O(n) for slicing
